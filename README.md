# 網軍大亨 (Cyberlord Tycoon)

## 專案說明

這是一個模擬網路輿情戰的策略養成遊戲。玩家將扮演一位網路輿情操控者，執行各種類型的任務，以提升自身的影響力並賺取虛擬貨幣。遊戲中充滿了駭客風格的視覺與設定，考驗玩家的策略規劃和資源管理能力。

## 技術關鍵字

- `pygame`: 用於遊戲開發，處理圖形、音效和使用者介面。
- `random`: 生成遊戲中的隨機事件，如任務生成、平台波動、雜湊衝突等。
- `sys`: 處理系統相關的操作，如退出遊戲。
- `pickle`: 用於遊戲進度的存檔與讀檔。
- `os`: 處理文件路徑和作業系統相關的操作。
- `time`: 控制遊戲的執行速度和時間相關事件。
- `threading`: 處理遊戲中可能出現的並行操作，例如後台任務執行或介面更新。
- 雜湊法 (Hashing):用於判斷使用者話語的正負向，這在遊戲中可能與輿情判斷相關。

## 功能

- **社群平台管理**: 模擬多個社群平台，每個平台有不同的風險與波動率。
- **多樣化任務**: 包含政治、商業、娛樂等不同類型的任務，提供不同挑戰與獎勵。
- **帳號系統**: 管理遊戲中的虛擬帳號，包含行動點數和封號風險。
- **道具與科技升級**: 玩家可購買道具和研發科技，以強化任務成功率、降低風險、提升影響力或降低成本。
- **存檔與讀檔**: 支援遊戲進度保存與載入。
- **駭客風格介面**: 採用深綠色調和終端機字體，營造獨特的遊戲氛圍。
- **時間模擬**: 遊戲時間流逝與事件發生。

## 環境需求

- Python 3.x (建議使用 Python 3.9.13)
- `pygame` 套件

## 安裝與執行

1 克隆專案:
git clone [您的專案 GitHub 連結]
cd [您的專案目錄]
2 安裝所需套件:
pip install pygame
(如果還有其他套件，請在此處列出，例如 pip install [其他套件])
3 執行遊戲:
python main.py  # 或您的遊戲主檔案名稱

## 遊戲設定

遊戲中定義了多項常數來控制遊戲體驗：

- **視窗大小**: `WINDOW_WIDTH`, `WINDOW_HEIGHT`
- **幀率**: `FPS`
- **顏色定義**: `WHITE`, `BLACK`, `BG_COLOR`, `PANEL_COLOR`, `BUTTON_COLOR`, `BUTTON_HOVER`, `TEXT_COLOR`, `GREEN`, `RED`, `GOLD` (皆為駭客風格配色)
- **任務類型顏色**: `TYPE_POLITICS`, `TYPE_COMMERCE`, `TYPE_ENTERTAINMENT`，以及對應的 `TYPE_COLORS` 字典。
- **社群平台設定**: `PLATFORMS` 字典，定義了每個平台的風險、波動率及顏色。
- **道具定義**: `ITEMS` 字典，包含道具名稱、價格及描述。
- **科技定義**: `TECHNOLOGIES` 字典，包含科技名稱、描述、基礎成本及最大等級。
